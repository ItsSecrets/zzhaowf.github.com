js的基本语法

一些常用知识点记录

Array的一些操作
  1. 检测是否为数组：Array.isArray(fruits), fruits为要检测的变量。
  2. 排序：
  1. slice基于原数组创建一个新数组，两个参数

  2. splice修改原数组，基于这个方法可以实现删除、插入、替换等操作

  3. Array的迭代，主要是every、filter、forEach、map、some

  1. 归并 reduce、reduceRight
Function
  1. aguments 类数组对象，包含着传入函数中的所有参数
  2. aguments.callee 一个指针，指向拥有这个aguments对象的函数（一些递归函数可能用的道）
  3. caller 函数内部用funName.caller可以调用到， 函数的调用者
  4. aguments.callee.caller 等价3
  5. apply
  6. call
  7. bind

    call()和apply()方法的作用相同，区别在于接受参数的方式不同。apply是直接传递aguments，而call是把参数直接传递

Number
  1. num.toFixed(2) 保留两位小数
String
  1. concat、slice、subString 会返回新串
  2. trim() 去除前后空格
  3. match
  4. replace
